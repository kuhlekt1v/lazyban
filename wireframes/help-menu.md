# Help Menu

## Wireframe

```
    ┌────────────────────────────────────────────────┐
    │ Keybindings                              [ESC] │
    ├────────────────────────────────────────────────┤
    │                                                │
    │  ─── Column Navigation ───                     │
    │  h, ←     Previous column                      │
    │  l, →     Next column                          │
    │                                                │
    │  ─── Card Navigation ───                       │
    │ ┌────────────────────────────────────────────┐ │
    │ │ j, ↓     Next card                         │ │
    │ └────────────────────────────────────────────┘ │
    │  k, ↑     Previous card                        │
    │  gg       First card in column                 │
    │  G        Last card in column                  │
    │  PgDn     Next group of cards                  │
    │  PgUp     Previous group of cards              │
    │                                                │
    │  ─── Card Actions ───                          │
    │  Enter    View card details                    │
    │                                                │
    │  ─── General ───                               │
    │  ?        Toggle this help menu                │
    │  q        Quit                                 │
    │                                                │
    │ ┌────────────────────────────────────────────┐ │
    │ │ Description:                               │ │
    │ │ Move focus to the next card in the current │ │
    │ │ column. Wraps to first card at bottom.     │ │
    │ └────────────────────────────────────────────┘ │
    │                                                │
    ├────────────────────────────────────────────────┤
    │ ↑↓/j,k: Navigate | ?, ESC : Close              │
    └────────────────────────────────────────────────┘
```

## Keybindings (Within Help Menu)
- `↑`, `k`: Navigate to previous keybinding
- `↓`, `j`: Navigate to next keybinding
- `?`: Close help menu
- `ESC`: Close help menu
